# RAHA KENYA B2B営業エンジン — 短尺動画マーケ支援の枠を能動的に埋める

> 目的: 既存の強み（18.6万フォロワー・年間4,400万売上・広告費ゼロで100万再生50本・媒体費0で14名採用）を
> "証拠"に、B2B短尺動画支援（初期50万＋月額45/55/85万）の契約社数を能動的に増やす。
> RAHAに一番欠けていて、Kuniの営業力が一番効く部分。
>
> 北極星: **MRR 1,000〜1,200万（15〜20社）＝ ARR 1.2〜1.4億 → 企業価値1億超**。
> 数値は `npm run saas -- --price 550000 --signups 2 --growth 0.06 --churn 0.02 --multiple 1.5` 等で試算可。
> ※[要確認]＝Kuniの実数で差し替えるプレースホルダ。

## 1. ICP（理想顧客像）— 誰に月55万を売るか

短尺動画の月額50万超を払えて、かつRAHAの強みが刺さる先:

| 優先 | セグメント | なぜ刺さるか |
|---|---|---|
| ★★★ | **採用に困っている中堅企業**（製造・介護・建設・地方優良企業） | RAHAは「媒体費0で採用難の壁を越えた」実績。採用×動画は予算が出やすい |
| ★★★ | **ニッチ/BtoB商材**（専門性が高く伝わりにくい） | 「ニッチを共感に変える翻訳力」がそのまま提案価値 |
| ★★ | **地方の老舗・1次産業/食品**（ストーリーはあるが発信下手） | 等身大の発信×ストーリーテリングが強み |
| ★★ | **海外展開/インバウンド狙いの企業** | ケニア×日本の越境発信の実体験が差別化 |

**避ける**: 動画を「本数」でしか見ない先、月20万以下しか出せない先（伴走モデルが割に合わない）。

## 2. ポジショニング（1行で刺す）

> 「**広告費ゼロで18.6万人を熱狂させた"現役クリエイター"が、御社の採用と認知を動画で変えます。**
> 綺麗なだけの代理店ではなく、ケニアの現場で毎日"最後まで見られる動画"を検証している実践者です。」

差別化の核（競合の制作会社にない3点）:
1. **自社で18.6万人・年4,400万を作った当事者**（外注・広告に頼らず）
2. **採用への直結実績**（媒体費0で150応募→14名）＝予算の出口が明確
3. **伴走型**（台本〜撮影〜編集〜改善まで二人三脚、丸投げOK）

## 5. 商談（15分診断）スクリプト

1. **現状ヒアリング**「今、認知と採用、どちらの課題が大きいですか？　今の動画/SNSは誰が回していますか？」
   → ほぼ「リソース不足/ネタ切れ/続かない」が出る ＝ 伴走型の刺しどころ
2. **痛みの言語化**「動画は"本数"より"最後まで見られるか"。綺麗な外注動画が伸びないのはそこです」
3. **証拠**「私たちは広告費0で18.6万人、100万再生50本、採用150応募。自分でやったから再現できます」
4. **提案（アンカリング）**「御社なら、まず{ベーシック55万}が適正です。本数を増やすならプロ85万、試すならライト45万」
   → **高い方（プロ）から見せて、ベーシックを"中央の安心"に置く**
5. **クローズ**「初期の戦略設計から入ります。まず6本回して反応を見る形と、最初から8本でいく形、どちらが御社に合いますか？」

### 想定反論と返し
| 反論 | 返し |
|---|---|
| 高い | 「採用1名の媒体費・紹介料と比べてください。私たちは媒体費0で14名採れています。動画は資産として残り続けます」 |
| 社内でやれる/外注済み | 「綺麗な動画は作れても"伸びる動画"は別物です。まず1本、企画から違いを無料でお見せします」 |
| 効果が不安 | 「毎月レポートと定例で数値化します。伸びなければ続ける必要はありません。だから初月から逃げません」 |
| ケニアと日本で大丈夫？ | 「むしろ越境でチームが回る仕組みを作ってきたのが強みです。撮影は現地ディレクションで対応します」 |

## 7. デリバリーの容量問題（営業と同時に潰す）

枠が埋まる前提で、撮影・編集の労働集約を**AIで軽くする**:
- 企画・台本: 過去の伸びた動画の型をAIに学習させ、企画案を量産
- 編集: テロップ・カット・横展開（YouTube Shorts/TikTok/Reels）をAI支援で多重化
- 横展開: 1撮影→複数尺・複数媒体に自動リサイズ

→ 1社あたり工数が下がれば、同じ人数で受けられる社数＝**MRRの天井**が上がる。

## 8. 正直な但し書き

- このキットは武器であって売上ではない。**数字が動くのは§4の1通目を実在の企業に送った瞬間から。**
- 最大の死因は集客ではなく**解約**と**デリバリー破綻**。営業で詰め込みすぎて品質が落ちると逆効果。
  → §7のAI化で容量を確保してから営業を踏む。
- 家族のチーム（代表・河野理恵）と**役割と取り分を最初に合意**すること。Kuniの貢献（営業×AI×資金）を明文化。
