## 0. この研修の「勝ち筋」＝棲み分けを最初に握る

注文住宅 × CADの研修で最も大事なのは、**「AIでCADの作図を置き換えるのではない」**とゴールを正しく設定すること。過去の建設・CAD系商談でも、ここを曖昧にすると「結局AIで図面が描けるの？」という期待値ズレで失注・不満につながっている。

ALTAは「**手書き間取り→3D自動変換**」「**プラン→各種図面・確認申請書類への自動変換**」「**KIZUKURI技術での構造伏図・梁せい計算**」「**宮川工機連携の自動木拾い・積算**」まで自社で完結する、すでにAIを内蔵した強力な設計エンジン。**作図・構造・積算の中核はALTAに任せる。**

研修で乗せる生成AI（ChatGPT / Claude / Gemini / 画像生成 / GPTs / NotebookLM）は、**ALTAの“前工程・後工程・周辺業務”を一気通貫で効率化する役割**。この棲み分けを1枚で示すのが研修の背骨になる。

```
   ┌──────────── 生成AIが効く領域 ────────────┐        ┌── ALTA（CAD）の中核 ──┐        ┌──── 生成AIが効く領域 ────┐
   集客・反響対応・ヒアリング・要望整理・           間取り作成 / 3Dパース /          資金計画の翻訳・提案書 /
   コンセプト立案・暮らしの提案文・                 立面・断面・展開図 /             確認申請チェック・施工指示・
   内装/外観イメージ画像・SNS/ブログ                構造伏図・梁せい計算 /            議事録・工程表・アフター案内・
                                                   積算・木拾い・各種申請図面         社内ナレッジQ&A・マニュアル
   ────────── 前工程 ──────────▶  ──── ALTAで設計・作図 ────▶  ────────── 後工程 ──────────
```

> **顧客への一言メッセージ**：「ALTAで描く品質はそのまま。AIは、**お客様と向き合う時間・考える時間**を生むために、その前後の“作業”を肩代わりします。」

## 1. 研修のゴール（受講後の状態）

1. 受講者全員が、**自分の担当業務で1つ以上のAI活用を“明日から”回せる**状態になる
2. **ALTAの前後工程**（集客〜提案〜引渡し〜アフター）で、AIが効く具体ポイントを地図として持っている
3. 自社専用の**「設計・接客アシスタントGPT」**と**標準プロンプト集**を、研修中に自分たちの手で作り上げる
4. 顧客個人情報・図面・見積データを扱う上での**リスク管理ライン**（入れていい情報／ダメな情報）を理解している

> ねらいは「ツールを知る」ではなく「**業務が変わる**」。各回に必ず“自社データで作る成果物”を置く。

## 3. 12時間カリキュラム（全6回×2時間）

各回：**座学3割（約35分）→ 自社データで手を動かす実践7割（約85分）→ 宿題（次回まで実務で1回試す）**。

### 第1回｜AIの全体像・棲み分け・リスク管理＋業務棚卸し
- **座学**：生成AIの今／ChatGPT・Claude・Geminiの使い分け／プロンプトの“型”（役割・前提・出力形式・例示）／**ALTAとAIの棲み分け図**の共有／注文住宅で外せないリスク管理（顧客個人情報・図面・見積データを「入れていい／ダメ」の線引き、ハルシネーション、著作権）
- **実践ワーク**：「2.の業務フロー×AI活用マップ」を**自社版に書き換える**。各自が自分の1業務を選び、その場でChatGPTに投げて効果を体感
- **成果物**：自社版 業務×AIマップ／各自の「最初に攻める1業務」決定
- **宿題**：選んだ1業務を実務で1回AIに任せてみる

### 第2回｜集客・ブランディング × AI（マッケンジーの世界観を増幅）
- **座学**：ブランドトーンの言語化／施工事例コンテンツの設計／画像生成（内装・外観イメージ、インフォグラフィック）／広告・LPコピーの作り方（※マッケンジーはGoogleブランド広告を運用 → ブランド指名の受け皿づくりが刺さる）
- **実践ワーク**：自社の施工事例を1件、**ブログ記事＋SNS投稿文＋サムネイル画像**まで一気通貫で生成。マッケンジーらしいトーン＆マナーをプロンプトに固定化
- **成果物**：施工事例コンテンツ1セット／自社ブランド用「トーン指定プロンプト」
- **宿題**：実際の施工事例でもう1本作って広報チャネルに出してみる

### 第3回｜接客・ヒアリング・提案 × AI ＋ ALTA連携①
- **座学**：お客様要望の構造化（要望→優先順位→ゾーニング）／ヒアリングシート設計／**ALTAの「手書き間取り→3D変換」スピード提案フロー**にAIをどう噛ませるか（AIで暮らしの仮説→手書き→ALTAで即3D）
- **実践ワーク**：仮想のお客様メモから、AIで**ゾーニング案＋暮らしの提案ストーリー＋提案書ドラフト**を作成（間取り・パースはALTA実機で）
- **成果物**：提案書テンプレート／ヒアリングシート
- **宿題**：直近の案件で提案文の下書きをAIで作ってみる

### 第4回｜資金計画・見積 × AI ＋ ALTA連携②
- **座学**：ALTAの積算・見積を**“顧客が分かる言葉”に翻訳**する／資金計画の比較提示／補助金・住宅ローン・税制の最新情報の集め方と注意点（要・一次情報確認）
- **実践ワーク**：サンプル積算データから、**顧客向け資金計画説明資料**と「見積根拠のやさしい説明文」を作成
- **成果物**：資金計画説明テンプレート／よくある質問への回答集
- **宿題**：実案件の見積で、説明文をAIに下書きさせる

### 第5回｜実施設計・現場 × AI ＋ 社内ナレッジのGPT化
- **座学**：法規・自社標準仕様のチェックリスト化／標準仕様書・指示書のドラフト／**過去図面・納まり・施工ノウハウをNotebookLM／GPTsでQ&A化**（“ベテランの頭の中”を全社で使う）／工程表・現場議事録・職人指示書
- **実践ワーク**：自社の過去資料（標準仕様・納まり集など）を読み込ませた**「設計・施工アシスタントGPT」を1つ構築**
- **成果物**：自社ナレッジGPTのβ版／現場用テンプレート（工程表・指示書）
- **宿題**：現場・設計で実際にナレッジGPTに質問してみて改善メモを取る

### 第6回｜業務プロセス自動化・定着・専用GPT仕上げ・ロードマップ
- **座学**：アフター/定期点検リマインド・口コミ/紹介依頼の仕組み化／議事録・マニュアルの自動化／**社内展開ルール**（誰がどう使う・情報管理・標準プロンプトの共有）
- **実践ワーク**：各自の**「自分専用AI活用キット」**（専用GPT＋標準プロンプト集）を完成・発表。社内標準プロンプト集をチームで統合
- **成果物**：**全社標準プロンプト集 v1**／各自の専用GPT／90日ロードマップ
- **修了状態**：全員が担当業務でAIを1サイクル以上回せる／自社GPTとプロンプト資産が残る

## 4. 用意するもの・受講環境

- **PC**：受講者各自が持参（実機ハンズオン前提）。ALTAが入ったPCがあると連携回（3〜5回）でリアル
- **アカウント**：ChatGPT有料版（推奨／会社共有1アカウントでも可）、Claude、Gemini。NotebookLMは無料で可
- **持ち込みデータ**：自社の施工事例、標準仕様書、納まり集、サンプル積算データ、ヒアリングシート現行版（※実顧客の個人情報はマスキングして使用）
- **会場**：対面なら執務スペースと別の会議室（助成金活用時の要件にも合致）。オンライン併用可

## 5. 成果イメージ（顧客に語るROI）

- **提案スピード**：要望整理〜提案書下書きの時間を大幅短縮し、ALTAの3Dスピード提案と合わせ「**その日のうちに提案**」を強化
- **設計・現場**：ベテランの納まり知識をGPT化し、若手の手戻り・確認待ちを削減
- **集客**：施工事例コンテンツの量産で、ブランド指名（Google広告）の受け皿を厚くする
- **全社**：議事録・マニュアル・アフター連絡の定型業務を自動化し、**人は接客・設計という付加価値業務に集中**
- 修了後は**専用GPT＋標準プロンプト集という“資産”**が社内に残り、研修が一過性で終わらない

## 6. 商談を前に進めるための事前ヒアリング項目（カスタマイズ用）

Transcript受領後、ここを埋めて各回ワークを確定する。

1. **受講者の職種構成・人数**（営業／設計／工事／広報／事務 各何名）とAI習熟度
2. **一番効率化したい業務トップ3**（社長／現場でズレることが多いので両方）
3. **ALTAの使用範囲**：誰がどこまで使っているか（営業はプレゼンのみ？設計まで？積算・伏図・申請図まで？）
4. **既存ツール**：Office / Google / 議事録 / SNS運用の現状
5. **データ持ち込み可否**：施工事例・標準仕様・積算サンプルを研修で使えるか（マスキング前提）
6. **集客の現状**：Google広告以外のチャネル、施工事例コンテンツの更新頻度
7. **実施形態**：対面／オンライン、6回×2時間 or 集中型、希望時期
8. **助成金**：人材開発支援助成金の活用希望有無（実質負担の試算を提示できる）
