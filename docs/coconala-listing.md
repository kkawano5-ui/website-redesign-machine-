# ココナラ出品キット（顔出しナシ・うさんくささゼロ設計）

そのまま貼って使える出品原稿一式。**顔を出さずに信頼感を作る**ことに全振りした構成。

---

## 0. 「顔出しナシ × うさんくさくない」の作り方（最重要）

占いのうさんくささは、ほとんど「①誰がやってるか分からない不安」と「②盛りすぎ・断定・煽り」から来る。顔を出さなくても、次の5つで信頼は作れる。

1. **顔の代わりに“手元”と“ロゴ”で人格を出す**
   - プロフィール画像＝ブランドロゴ（清潔な明朝＋月と手のシンボル）。
   - ギャラリーに「鑑定中の手元（命式を書く手・PCで鑑定書を作る手）」の写真を入れると、顔なしでも“人がやってる安心感”が出る。AI生成の手元イラストでも可。
2. **サンプル鑑定書（実物PDF）を必ず公開する**
   - ギャラリーに実際の鑑定書PDFの数ページを画像化して載せる。「中身が見える」=最大の信頼。`scripts/fortune/pdf.js` の出力をそのまま使える。
3. **手順と納期を具体的に書く**（曖昧さがうさんくささの正体）
   - 「購入→情報送付→○日以内に鑑定書PDFをお届け」と明記。
4. **盛らない・断定しない・煽らない**
   - 「必ず当たる」「あなたは呪われている」系はNG。エンタメ・自己理解と明記。不安商法をしない姿勢そのものが差別化になる。
5. **再鑑定・質問対応の保証**
   - 「分かりにくければ無料で1回質問対応します」など、逃げない姿勢を見せる。

> ココナラ規約上も、断定的・不安を煽る表現、医療/健康/寿命の断定はNG。守ることが結果的に信頼に繋がる。

## 4. サービス説明（本文・そのまま貼れる）

```
ご覧いただきありがとうございます。Luna Mano 鑑定室です。

このサービスは、
◆ 四柱推命（生年月日から導く“生まれ持った設計図”）
◆ 手相（手のひらの写真から読む“今のあなた”）
この二つを「重ねて」読み解く複合鑑定です。

生年月日からは、あなたの本質・気質・運の流れ（命式）を割り出します。
手相からは、いま現れている状態や、選んできた生き方を読みます。
両方が同じ方向を指す点・補い合う点をまとめ、1冊の鑑定書（PDF）にしてお届けします。

【こんな方へ】
・自分の強みや本質を言葉にしてほしい
・占いは好きだけど、不安を煽られるのは苦手
・恋愛/仕事/全体運について、前向きなヒントがほしい

【大切にしていること】
・断定や不安を煽る表現はしません。あくまで自己理解のための鑑定です。
・短所も「伸びしろ」として前向きにお伝えします。
・鑑定書は読みやすいPDFで、後から読み返せます。

まずはお気軽にメッセージからご相談ください。
```

⑤ 手のひらの写真（利き手・手のひら全体／撮り方は下記）

▼手のひら写真の撮り方
・明るい場所で（自然光が理想／フラッシュの白飛びは避けて）
・指を軽く広げ、線がはっきり見えるピントで画面いっぱいに
・基本は利き手（両手いただけるとより詳しく読めます）

情報が揃い次第、鑑定書（PDF）を○日以内にお届けします。
```

## 7. よくある質問（FAQ・出品文末尾に掲載）

```
Q. 出生時刻が分かりません。鑑定できますか？
A. 可能です。生年月日だけでも鑑定できます。時刻が分かると、より詳しくお読みできます。

Q. 手相の写真がうまく撮れているか不安です。
A. 撮り方のコツをご案内します。不鮮明な場合は撮り直しをお願いすることがあります。

Q. 当たりますか？／悪い結果は出ますか？
A. 占いは自己理解を深めるエンターテインメントとしてご提供しています。不安を煽る表現や、健康・寿命の断定はいたしません。短所も前向きにお伝えします。

Q. 顔出し・通話は必要ですか？
A. 不要です。メッセージのやり取りだけで完結します。

Q. 個人情報は大丈夫ですか？
A. お預かりした写真・生年月日は鑑定のみに使用し、無断で二次利用しません。
```
