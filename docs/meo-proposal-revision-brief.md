## ④ 貼り込み用コピー（そのままスライドへ）

### 【スライド6・竹の説明差し替え】
```
竹 スタンダード（月29,800円）★イチオシ
投稿は「企画・文案・画像選び・掲載」まで丸ごとお任せ。
本業が忙しくても、Googleマップが“止まっている店”になりません。
・月4回投稿（企画〜掲載まで代行）
・口コミ返信文の作成（返信はワンタップ）
・写真の選定・整理／競合の簡易チェック
・月次レポート＋翌月の改善提案
```

### 【スライド8・竹を勧める理由の筆頭】
```
① 投稿を、丸ごとお任せできる
「投稿する時間がない」を仕組みで解決します。企画・文案・画像選びから
Googleへの掲載まで私たちが実施。あなたは承認するだけ。だから、
忙しい月でもマップが“最近やっていない店”に見えることがありません。
```

### 【新規スライド・オンボーディング面談の価値】
```
はじめに、30〜60分の面談で「運用の方針」を一緒に決めます

・お店の強み・NG表現・伝えたいことをヒアリング
・投稿のテーマ方針、口コミへの向き合い方を一緒に確定
・以後の運用は、この“決めた方針”の中で自動的に回ります

丸投げでも、あなたの意図からズレない。
最初にきちんと握るから、毎月の確認は最小限で済みます。
```

### 【新規 or スライド9内・正直にできないこと】
```
私たちが「しないこと」を、正直にお伝えします

・順位・問い合わせ数・売上の“保証”はしません
  （Googleの仕組み上、誰にもできません。数値は毎月正直に開示します）
・自動で勝手に公開はしません
  AIが下ごしらえし、人が薬機法・表現・トーンを確認してから公開します
・サクラ・対価と引き換えの口コミは、ご希望でも一切行いません
  それが、2026年のポリシー厳格化からあなたの口コミ資産を守る方法です
```

### 【スライド5内・安心の仕組み（human-in-the-loop）】
```
AIの速さ × 人の目

日々の投稿・口コミ返信の下ごしらえはAIが自動で。
公開の前に、必ず人が薬機法・業種ルール・トーンを確認します。
「速いのに、雑じゃない」を仕組みで両立します。
```
