## 経営計画書ドラフト（様式2 相当・たたき台）

### 1. 企業概要
当社は2020年設立の法人。代表1名体制で、IT・コンサルティングを基盤事業としつつ、2026年より訪日外国人（英語圏アッパーミドル層）向けのプレミアム体験事業を本格展開する。提供は ①少人数体験ツアー ②プライベート貸切・限定アクセス ③bespoke旅程設計＋家族フォト＋コンシェルジュ の3本柱。代表は英語ネイティブ水準の対応力を強みとし、企画から催行・接客までを一貫して担う。

### 2. 顧客ニーズと市場動向
訪日インバウンドは回復・拡大基調にあり、特に**高単価・少人数・本物志向**の体験需要が伸びている。一方、英語で深く対応できる「地方・hidden gems」「酒・ウイスキー」「アニメ聖地（英語解説）」等の専門体験は供給が少なく、富裕層の満足度・客単価・再来訪に直結する空白市場が存在する。

### 3. 自社の強み
- 英語ネイティブ水準の企画・接客（一次情報での体験設計）
- 予約困難店・限定アクセスを組み込むbespoke対応力
- IT・コンサル基盤による業務のデジタル化・データ活用適性

### 4. 経営方針・目標
直販比率を高め（OTA手数料の圧縮）、体験→プライベート→bespoke/フォトへ顧客を引き上げる多層モデルで、初年度 粗利1,300万〜3,000万円規模を目指す。
