# 補助金 取得プラン（社長1人・埼玉県・法人）

> 最終更新: 2026-06-18
> 注意: 金額・補助率・締切は変動します。**着手前に必ず各制度の最新「公募要領」で確認**してください（本書は各所に出典URLを記載）。

## 0. 御社の前提（補助金審査で効く事実）

| 項目 | 内容 | 補助金での意味 |
|---|---|---|
| 事業形態 | 法人（2020年設立） | ほぼ全制度で「法人」として申請可 |
| 決算 | 毎年申告あり | **直近決算書を出せる＝主要補助金の必須要件をクリア** |
| 規模 | 社長1人・従業員なし | 「小規模事業者（商業・サービス業は従業員5名以下）」に該当 → 持続化補助金OK |
| 事業内容 | ①IT・コンサル ②訪日インバウンド向けプレミアム体験 | 両方を「労働生産性向上」「販路開拓」で説明できる |
| 所在地 | 埼玉県 | 国の制度＋埼玉県・市区町村の独自補助金も対象 |
| これまで | ほぼ休眠 → 2026年から本格稼働 | **「創業型」は対象外**（創業後1年以内要件）。一般枠で申請する |

### 重要な前提の確認事項（申請前に潰すべき2点）
1. **直近の決算は「ほぼ休眠」でも申告済みであること**（赤字・売上ゼロでも申請自体は可能な制度が多いが、事業実態の説明が必要）。
2. **市区町村の自治体補助金は「所在地の市」固有**。会社所在地の市区町村名が確定したら、その市のサイトを確認（本書末尾に探し方）。

## 1. 適合する補助金 ショートリスト（推奨順）

| 順位 | 制度 | 適合度 | 主な使い道 | 補助上限 / 補助率 | 次の締切（2026） |
|---|---|---|---|---|---|
| ★1 | **デジタル化・AI導入補助金2026（旧IT導入補助金）** | ◎ | 予約システム・CRM・会計・業務効率化ソフト | 通常枠 最大450万 / 1/2（条件で2/3） | **通年・1〜2ヶ月ごと**（公募中） |
| ★2 | **小規模事業者持続化補助金〈一般型〉** | ◎ | 多言語Web・LP・広告・パンフ・撮影機材・展示会 | 50万（特例で最大250万）/ 2/3 | 第20回 受付 11/5〜12/15 |
| 3 | **埼玉県 中小企業省力化支援事業** | ○ | 省力化につながる設備・ロボット・AI機器 | 1,000万（賃上げで1,200万）/ 2/3（賃上げ4/5） | 県サイトで要確認 |
| 4 | **キャリアアップ助成金 / 人材開発支援助成金** | ○（採用時） | ガイド・スタッフを雇う／非正規→正社員化 | コースにより数十万〜 | 通年（雇用後に申請） |
| 5 | 観光庁・インバウンド系（高付加価値化・地域づくり 等） | △ | 多言語化・受入環境整備 | 事業による | 公募時期限定 |
| 6 | 市区町村の独自補助金（HP制作・販路開拓・DX 等） | ○ | 所在地次第 | 数十万規模が多い | 自治体次第 |

> ★1・★2 を本命の2本柱として進めます（使い道が重複せず、両取り可能）。
> ※ 同一経費を複数補助金で二重取りはできません。経費を「IT＝★1」「広告・Web・機材＝★2」と振り分けます。

### 御社の使い道（カード決済・Meta広告・PC）の振り分け
| 使いたいもの | 取る補助金 | 注意 |
|---|---|---|
| カード決済（予約/POSソフト） | ★1 デジタル化・AI導入補助金 | 決済手数料そのものは対象外。ソフト導入費が対象 |
| Meta広告（FB/IG） | ★2 持続化補助金（広告費） | IT補助金では広告は対象外 |
| PC等のハード | ★1 の**インボイス枠**（ソフトとセット時のみ） | PC単体は不可。持続化では「汎用品」で対象外 |

## 2. 私（Claude）の作業スコープ ＝「申請までやる」の中身

### ✅ 私がやること（このリポジトリ内で完結）
- 適合補助金の選定と戦略設計（本書）
- **申請書類のドラフト作成**：経営計画書 / 補助事業計画書 / 労働生産性向上の説明 / 事業の必要性
- 必要書類チェックリスト・スケジュール表
- IT導入補助金の「ITツール候補」「IT導入支援事業者」の探し方の整理
- 提出前レビュー（要件の取りこぼし確認）

### ⚠️ 御社本人がやる必要があること（私が代行できない）
- **gBizIDプライムの取得**（印鑑証明・登記情報が必要／本人申請）
- **電子申請（Jグランツ / 各事務局ポータル）へのログインと送信**
- **IT導入支援事業者・ITツールの最終選定と契約**（★1）
- **商工会議所での「事業支援計画書（様式4）」発行依頼**（★2、対面/オンライン）
- 見積書・決算書・本人確認書類など**実データの提出**
- 申請内容の最終承認（事実と相違ないことの確認）

> つまり私は「中身を全部書いて、貼り付け・送信するだけの状態」まで持っていきます。送信ボタンは御社が押す形になります（公的申請は本人認証が必須のため）。
