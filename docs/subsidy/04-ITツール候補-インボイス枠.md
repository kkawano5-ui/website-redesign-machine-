> スケジュール: https://it-shien.smrj.go.jp/schedule/
> ⚠️ 登録ツール・支援事業者は年度内も入れ替わる。本書の製品名は「**候補（要・現行登録確認）**」。最終確定は支援事業者と公式検索で。

---

## 0. インボイス枠のルール（ここが設計の肝）
- 対象機能：**「会計」「受発注」「決済」**
- **2機能以上**を含む → 補助額 **350万円以下** ／ **1機能のみ** → 補助額 **50万円以下**
- 補助率：1/2（小規模事業者は賃上げ等で最大4/5）
- **ハード（PC・POS・決済端末）はインボイス枠＋対応ソフトとセットの時のみ対象**（PC類 補助上限10万・POS等 20万）

### → 御社の最適設計
**「会計」＋「決済」の2機能**を満たすソフト構成にして **350万枠**に乗せる。
その上で **業務用PC＋カード決済端末/POS** をハードとしてセットで拾う。

## 1. 機能別 候補ソフト（すべて「要・現行登録確認」）

### ① 会計（インボイス対応）＝枠の主軸
| 候補 | 特徴 | 備考 |
|---|---|---|
| **freee会計** | インボイス・請求・受発注まで一体。IT補助金の登録実績が多い | 受発注/請求も使えば「会計＋受発注」で2機能化しやすい |
| **マネーフォワード クラウド** | 会計＋請求書＋債務支払。決済連携も | 同上 |
| **弥生会計 オンライン** | 定番・サポート手厚い | |

### ② 決済つき 予約／POS ＝「決済」機能＋カード決済を満たす
| 候補 | タイプ | 御社適合（インバウンド体験） |
|---|---|---|
| **スマレジ（＋PAYGATE等）** | POS＋決済 | ハード（タブレットPOS）も補助実績多く、PC/POSを拾いやすい |
| **Airレジ＋Airペイ（リクルート）** | POS＋決済 | カード/QR/タッチ決済に広く対応 |
| **Square** | POS＋決済端末 | 端末安価・多通貨/海外カードに強い＝訪日客向き |
| **STORES 決済／予約** | 予約＋決済 | ネット予約＋カード決済を一体運用 |

### ③ 多言語 予約システム（カード決済つき・任意で追加）
| 候補 | 特徴 |
|---|---|
| **RESERVA** | 予約＋多言語＋カード決済。アクティビティ/体験向き |
| **Coubic（クービック）** | 体験・レッスン予約＋決済 |
| **STORES 予約** | 予約＋事前カード決済 |

> ※ 予約システムの「予約機能」自体はインボイス枠の3機能（会計/受発注/決済）には数えない。**枠の2機能は「会計＋決済」で満たし**、予約・多言語は付加価値として導入する整理が安全。

## 2. 御社向け おすすめ構成（たたき台）

**構成案：会計＋決済の2機能 → インボイス枠350万枠**
1. **会計**：freee会計 or マネーフォワード（インボイス対応／請求・受発注も使い「会計＋受発注」を確実に2機能化）
2. **決済**：Square or Airペイ（訪日客の海外カード・タッチ決済に対応）
3. **ハード（セット）**：業務用PC（補助上限10万）＋ カード決済端末/タブレットPOS（補助上限20万）
4. （任意）**多言語予約**：RESERVA等で予約〜事前決済を自動化

> 御社は社長1人・IT/コンサル＋インバウンド体験。規模的にIT投資総額は数十万〜百数十万円規模が現実的で、補助はその1/2（小規模＋賃上げで最大4/5）。「PCを安く買う」より「会計・決済・予約のDX一式を半額で入れる」という位置づけが妥当。

## 3. 進め方（A→申請まで）
1. **公式ツール検索**で上記候補の**現行登録**を確認 → 1〜2社に絞る
2. その製品の **IT導入支援事業者（ベンダー）に問い合わせ**（補助金申請を一緒にやってくれる）
3. 支援事業者と **インボイス枠/通常枠どちらで出すか**を確定（決済+会計+PCなら**インボイス枠**）
4. 見積取得 → 交付申請（事業計画値は `03-事業計画値-労働生産性3案.md` を使用）
5. **交付決定後**に契約・発注（※決定前発注は失格）

### 直近スケジュール
- 通常枠 第3次締切：**2026/7/21 17:00**（インボイス枠の締切回は公式スケジュールで要確認）
- まず **gBizIDプライム**（取得2〜3週間）と **SECURITY ACTION**（無料）を即着手

## 4. 御社→私に欲しい情報
- 候補のうち**どれを使いたい/既に使っているもの**があるか（freee/MF、決済はSquare/Airペイ等）
- 想定IT投資の**ざっくり総額**（→ 補助額の見込みを出します）
- これで構成と経費明細（`01`の経費表）を実数で確定し、申請の経費入力欄に入る形に清書します。
