# 補助金 目標 200万円 — 逆算設計（★1 デジタル化・AI導入補助金 インボイス枠）

> 結論：**200万円の補助は獲れる**。ただし補助金は「実支出の一部補助＝自己負担あり・後払い」。
> 200万円もらうには、補助率に応じて **250万〜400万円の対象経費**が必要。

---

## 1. いくら使えば200万円もらえるか（逆算）

| 補助率 | 必要な対象経費 | 自己負担（差額） | 適用イメージ |
|---|---:|---:|---|
| 1/2 | **400万円** | 200万円 | 通常の補助率 |
| 2/3 | **300万円** | 100万円 | 賃上げ等の上乗せ要件を満たす場合 |
| 3/4 | 約267万円 | 約67万円 | インボイス枠 下位区分など |
| **4/5** | **250万円** | **50万円** | 小規模事業者＋賃上げ要件（最も自己負担が軽い） |

> インボイス枠の補助上限は **350万円**（会計/受発注/決済の2機能以上）なので、200万円は枠内。
> ✅ **狙い目＝補助率を上げて、必要支出と自己負担を圧縮する**。御社は「小規模事業者」なので、**賃上げ要件**を満たせれば4/5（自己負担たった50万円で200万円獲得）が理想形。

## 3. 補助率を上げる「賃上げ要件」について（4/5を狙う鍵）
- 一般に、補助率上乗せには **給与支給総額の増加** や **事業場内最低賃金** の引上げ表明などが必要（具体水準は公募要領）。
- 御社は今年から本格稼働＝**役員報酬・採用で人件費が増える局面**なので、賃上げ計画は描きやすい。
- ⚠️ ただし **「役員のみ・従業員0」の場合に賃上げ要件をどう満たすか**は事務局に要確認（従業員を雇う前提なら満たしやすい）。
- → `03-事業計画値` の **B本線（2年目以降に1〜2名雇用）** と整合させると、賃上げストーリーが一貫する。

## 4. ⚠️ 必ず理解しておくこと（お金の流れ）
1. **後払い**：先に自分で全額（例：340万円）支払い → 実績報告 → あとから補助金（例：200万円）入金。**つなぎ資金が要る**。
2. **自己負担は必ず発生**：200万円もらうにも 50万〜200万円の自己負担。
3. **交付決定“前”の発注・契約・支払いは対象外**（順番厳守）。
4. **実績報告・効果報告が必須**：導入したツールを実際に使い、生産性向上を報告する義務。

> つなぎ資金が不安なら、日本政策金融公庫の融資と組み合わせる手もある（別途相談可）。
