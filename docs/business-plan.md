## 2. ユニットエコノミクス（1催行の粗利）

```
小グループ(6名×¥25k=¥150k)
 直販(SNS)経由 : ¥150k − 原価¥42k − ガイド¥20k            = 粗利 ¥88,000   ← OTA手数料ゼロ
 OTA経由       : ¥150k − OTA22%¥33k − 原価¥42k             = 粗利 ¥75,000（家族催行）
 OTA×外注      : ¥150k − OTA¥33k − 原価¥42k − ガイド¥20k    = 粗利 ¥55,000
プライベート(直販¥90k) : ¥90k − 原価¥15k                    = 粗利 ¥75,000
```
**目安：1催行あたり粗利 ≒ ¥55,000〜88,000。直販比率が上がるほど粗利率UP。**

## 6. 投資・コスト

- **初期投資（〜¥150万）**：撮影機材/制作、予約LP（マシン）、検証スプリント（OTA/広告/インフルコラボ ¥30〜80万）。
- **月次固定費（〜¥50〜100万、粗利連動で拡大）**：SNS制作、広告（指名/補完のみ）、ツール/決済、ネイティブQA。外注ガイドは催行連動（粗利に織込み済）。
- **家族＝中核労働**で固定人件費を最小化。粗利が出てから増員。
- **法人化推奨**（社保・節税・信頼）。

## 7. マイルストーン

- **90日**：検証スプリント完了→経路別CPA実数→本命1軸＋看板2商品を確定。OTAレビュー10+、SNS初速、初の直販予約。
- **6ヶ月**：損益分岐。外注ガイド体制。SEO記事30本。月催行20+。
- **12ヶ月**：**粗利¥22M（本線）着地**。レビュー200+、直販30%、紹介エンジン稼働。
- **24ヶ月**：粗利¥62M。多拠点/3軸展開、外注網でキャパ拡張。

## 8. 90日ゲート（GO/PIVOT判定）

検証スプリント後、`sns-acquisition-playbook.md`の判定マトリクスで意思決定：
- **型B(CPA)成立 × 型A(SNS)伸びた** → 両エンジン併走（最速）。本計画通り。
- **型Bのみ** → 事業者主導でOTA/SEO中心に。
- **型Aのみ** → クリエイター主導（立ち上げ長期化を覚悟）。
- **どちらもダメ** → 軸/商品を変えて再試行（大型投資はしない）。
