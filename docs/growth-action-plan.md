# 成長・行動計画（Growth Action Plan）— ゼロ→¥1億

> 親ドキュメント：`docs/luxury-concierge-strategy.md`（DR統合戦略）。本書はそれを**実行タスク単位**に落とした行動計画。
>
> 一文方針：**「ハードな“アクセス（供給）”を握り → 信頼できるアドバイザー/コンシェルジュ経由で配り → 初回を完璧に届けてレビュー/紹介で複利を回す」**。消費者向け広告では勝たない。

---

## 0. 成長の方程式（これだけは暗記）

```
売上 = 有効リード数 × 成約率 × 1件あたり粗利 × (1 + 紹介・リピート係数)
```

- ¥1億の到達式（自社粗利ベース／戦略メモ §3）：**¥50万 × 200件** ／ ¥100万 × 100件 ／ ¥200万 × 50件。
- 成熟時で**週2〜4件**の成約。ゼロからは無理なので**3年スパン**で積み上げる（下記ランプ）。
- 動かすレバーは4つだけ：**①供給（アクセス）②配荷（販路）③転換（ファネル）④複利（紹介・リピート）**。本計画は全フェーズでこの4つを進める。

### 売上ランプ（現実的な目安）
| 期間 | フォーカス | 成約件数/年 | 粗利規模(目安) |
|---|---|---|---|
| **Year 1** | 証明（供給構築＋初回実績＋ファネル確立） | 20〜40件 | ¥1,000〜2,500万 |
| **Year 2** | 配荷（アドバイザー/ホテル網に接続） | 60〜100件 | ¥4,000〜7,000万 |
| **Year 3** | 拡張（ホスト体制・名物商品・リテイナー） | 150〜200件 | **¥1億射程** |

> ¥1億は90日目標ではない。90日の目標は「**どのテーマなら高単価で選ばれるかの証明**」（戦略メモ §6）。

## フェーズ0 — 「予約を受けられる状態」を作る（Week 1〜4）

“売る前に、確実に届けられる体制”を先に固める。ここを飛ばすと初回で事故り、複利が逆回転する。

### 0-A. 楔の確定（1テーマ・1都市）
- [ ] 主戦場を**「東京：予約困難ダイニング＋夜の私的二軒目」**に固定（自分が実地で品質保証できる範囲＝戦略メモ §6）。
- [ ] 看板となる**A Day（1日フルホスト）**の標準コースを1本設計（夕食→文化→夜の流れ）。

### 0-B. 法務・信用（初手で固める＝戦略メモ §5）
- [ ] 旅行業 vs 旅行サービス手配業の線引きを専門家に1回相談。**自社契約する範囲／登録旅行会社に寄せる範囲**を決定。
- [ ] 事業主体（屋号/法人）・利用規約・キャンセルポリシー・賠償/保険・免責を整備。
- [ ] LP表記を事実準拠に（「登録旅行業者と提携」等、保有しない資格は主張しない）。

### 0-C. 決済・お金の流れ
- [ ] Stripe（多通貨）＋ Wise でデポジット→請求→実費精算のフローを用意。FX/手数料の扱いを規約に明記。
- [ ] 料金v1（Access ¥15k〜／A Day ¥200k〜／Journey 20%・最低¥50万）で見積テンプレを作成。

### 0-D. 供給（アクセスの堀）★最重要
- [ ] **供給先20〜30件**と関係構築：予約困難店・バー・旅館・工房/職人・通訳ガイド・専用車。
- [ ] 非公開の**サプライヤー台帳**（連絡先・リードタイム・席数/枠・手数料/コンプ・品質メモ）を作成。
- [ ] 各供給先の「**何日前までに何が必要か**」を確定（直前予約事故の予防）。

### 0-E. 提供オペレーション（初回事故ゼロ化）
- [ ] ゲスト・インテークフォーム（日程/人数/食物アレルギー/関心/予算/移動）。
- [ ] ホスト・ブリーフィング雛形＋当日ランブック（タイムライン・連絡網・緊急時）。
- [ ] 提案書テンプレ（写真＋行程＋料金内訳＝フィー/実費の透明表示）。

### 0-F. 受注導線
- [ ] LPの問い合わせ → CRM（Notion or HubSpot無料）→ **返信SLA 12時間以内**。
- [ ] 問い合わせ〜初回返信〜提案〜デポジットの**ステージ管理**を1枚で可視化。

**フェーズ0の完了条件**：「明日、富裕層から1件来ても、品質・法務・決済・体験すべて事故なく回せる」。

## フェーズ1 — 最初の10件で「実績」を作る（Month 1〜3）

需要は**温かいチャネル**から取る。広告に金を使わない。狙いは“売上”より**信頼資産（レビュー/事例/写真）**。

### 1-A. ソフトローンチ（温かい順に）
- [ ] 既存の人脈・在日富裕層/駐在コミュニティ・酒LPの既存流入に告知。
- [ ] **ファウンディング・ゲスト**として5〜10名を手選び。看板A Dayを提供 → 代わりに**レビュー・写真・短尺動画・推薦文**をもらう。

### 1-B. 信頼資産の収集（複利の燃料）
- [ ] Google/Trustpilot等で**★5レビュー5件以上**。
- [ ] **写真付き事例（ケーススタディ）1本**（行程・体験・声）。
- [ ] アドバイザー配布用**1枚もの（トレード・ワンページャー）**の素地に流用。

### 1-C. ファネルを締める（毎回同じ型）
```
問い合わせ → 20分ディスカバリーコール → ベスポーク提案 → デポジット
→ 完璧に提供 → レビュー依頼 → 紹介のお願い
```
- [ ] この7ステップを台本化。**コール→提案→デポジット**の各転換率を記録（ベースライン化）。

**フェーズ1の完了条件**：有料成約10件・事例1本・レビュー5件・**成約率の基準値**が出る。

## フェーズ2 — 配荷エンジンを接続する（Month 3〜9）★伸びの本丸

富裕層旅行は**B2B2Cで配る**（戦略メモ §4）。3チャネルを並行で立てる。

### 2-A. ラグジュアリー旅行アドバイザー（最重要チャネル）
- [ ] **トレード・キット**作成：ワンページャー＋レート/コミッション表＋事例＋FAM（視察）オファー。
- [ ] ターゲット：Virtuoso／Serandipians 加盟アドバイザー、米国の日本特化アドバイザー、海外DMC。
- [ ] アウトリーチ：温かい紹介→コールド。商談イベントに接続（**ILTM Cannes/Asia Pacific・Connections Luxury・PURE・Virtuoso Travel Week**、JNTOラグジュアリー企画）。
- [ ] 1人の優良アドバイザーは継続送客源。**FAMで実体験させて“語れる”状態**にする。

### 2-B. 高級ホテルのコンシェルジュ（国内連携）
- [ ] 東京/京都の5★（Aman・Bulgari・Four Seasons・Janu・星のや 等）のコンシェルジュデスクと**送客/コミッション提携**。
- [ ] 「自分たちにしか出せない私的アクセス」を実演 → デスクの“切り札”に採用される。

### 2-C. 直販（指名を取る）
- [ ] 高意図の英語SEOページを数本（例：「how to get into impossible Tokyo restaurants」「private omakase access」）。検索の刈り取り＋アドバイザーへの信用補強を兼ねる。
- [ ] 短尺SNS（`sns-acquisition-playbook.md`）で食/hidden gems/夜を回し、**本命の高単価商品へ送客**。
- [ ] 厳選PR：Robb Report／Condé Nast Traveler／Travel+Leisure／AFAR にピッチ。

### 2-D. 紹介プログラム
- [ ] 過去ゲスト向けの紹介インセンティブ（次回フィー割引 or 特別体験）を設計。

**フェーズ2の完了条件**：**継続送客してくれるパートナー3〜5社**、パートナー経由の成約が出る、パイプラインが積み上がる。

## フェーズ3 — ¥1億へスケールする（Month 9〜24）

品質を落とさず件数と単価を上げる。

- [ ] **ホスト/ガイドのベンチ**（案件単位の契約制）を構築＋**品質基準・研修・QA**を標準化。
- [ ] **名物・多日程パッケージ**（命名された旅程）を商品化 → 平均単価をJourney帯へ引き上げ。
- [ ] **リテイナー/メンバーシップ**（リピートHNW向け継続課金＝第1層の固定フィー）。
- [ ] **地方供給の拡張**（金沢・瀬戸内・紀伊・東北・白馬/ニセコ）で“自分では組めない日本”を差別化。
- [ ] **ユニットエコノミクス再計算**：実ホスト原価・稼働日数・成約率を入れて§3の¥1億式を自社現実値で再試算（戦略メモの宿題）。

## KPIダッシュボード（PV/フォロワーは見ない＝戦略メモ §4）

毎週/毎月、これだけ追う：

| 指標 | 何を見る |
|---|---|
| 有効問い合わせ数 | TOFUの質（量ではなく富裕層度） |
| ディスカバリーコール数 → 提案数 → 成約数 | ファネル各段の転換率 |
| **成約率** | コール→成約の歩留まり |
| **1件あたり粗利** | 単価の健全性（フィー＋実費モデル） |
| **パートナー経由比率** | 配荷エンジンの効き |
| **紹介・リピート率** | 複利の効き（最重要の先行指標） |
| 供給先の再受入率 | 供給関係の健全性 |
| NPS / レビュー★ | 体験品質 |

## 勝敗を分ける3点（迷ったらここに戻る）

1. **模倣困難なアクセス（供給の堀）** — 「自分では予約できない/組めない」を本当に握る。
2. **信頼されるアドバイザー/コンシェルジュ経由の配荷** — 消費者広告ではなくB2B2C。
3. **初回の完璧な提供 → レビュー/紹介** — 複利ループ。1件目を雑にやると全部止まる。
