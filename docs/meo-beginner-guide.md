# MEO事業 素人向け完全ガイド（営業できる・事故らない）

> **これは何**：MEO事業がはじめての人でも、(1)お客様に自信をもって説明でき、(2)Googleポリシー・薬機法のリスクを避け、(3)お客様の課題にちゃんと刺さる、ようになるための1枚ガイド。
> **元資料**：`meo-service-detail.md`（サービス詳細）/ `meo-escalation-rules.md`（安全ルール）/ `meo-operating-model.md`（運用）/ `meo-contract-revision.md`（契約）。困ったら元資料に戻る。

---

## 0. まず結論（30秒で言えるMEOの説明）

> 「MEOは、**Googleマップで比較されたときに"この店にお願いしたい"と思ってもらう状態をつくる仕事**です。順位を上げる魔法ではなく、写真・口コミ・投稿・情報を整えて、**選ばれる受け皿**をつくります。」

この一言が全ての土台。**「順位を上げます」と言わない**のが、素人が最初に覚えるべき唯一のルール。

---

## 1. そもそもMEOって何？なぜお客様は困ってる？

### お客様（店舗）に起きていること
1. お客様は「大宮 ネイル」みたいに**地域名＋業種**で検索する
2. Googleマップに数店がズラッと並ぶ
3. 見込み客は数十秒で**写真・口コミ・返信・情報の新しさ**を見比べる
4. 「ここよさそう」と思った1店に電話・予約する

→ **表示されても、比較で負けたら1円にもならない。** これが店舗の"もったいない"。

### だからMEO＝「受け皿づくり」
順位（Googleが勝手に決める）ではなく、**比較されたときの見え方**を整える。ここは我々がコントロールできる。この考え方が、後の「順位保証しない」という誠実な営業につながる。

---

## 2. 我々が「実際にやれること」（素人向けに超シンプル化）

8つのサービスがあるが、素人はまず**この3つが主役**と覚える：

| 主役サービス | ひとことで | お客様のメリット |
|---|---|---|
| **① 投稿代行** | Googleマップに定期的に投稿する | 「最近やってない店」に見えない＝活きてる感 |
| **② 口コミ返信** | 口コミに丁寧に返信する | 「ちゃんとしてる店」＝信頼される |
| **③ 現状診断＋情報整備** | プロフィールを整える | 比較で負けない土台 |

残り5つ（口コミ獲得導線・写真運用・NAP統一・順位計測・月次レポート）は**"おまけの強化パーツ"**。最初は主役3つだけ理解すればOK。

### 「どこまでやれるか」の正直な答え（お客様に聞かれる3大質問）

| お客様の質問 | 正直な答え |
|---|---|
| 「投稿、代わりにやってくれるの？」 | **はい。竹プラン以上なら、企画から掲載まで丸ごと代行。** あなたは承認するだけ |
| 「口コミの返信も？」 | **文案は全プラン作ります。返信の実行まで代行は松プラン。**（竹はワンタップで貼るだけ） |
| 「サボらず続けてくれる？」 | **AIが下ごしらえ→人が確認→掲載、を仕組みで回すので止まりません。** ただし成果（順位）は保証しません |

## 3. 【最重要】リスクを避ける — これを破ると客の店が消える

> MEOで一番怖いのは「効果が出ない」ことではなく、**やり方を間違えてお客様のGoogleアカウントが停止（BAN）されること**。2026年4月からGoogleは厳格化し、違反すると**口コミ全削除・プロフィール停止**になる。だから"やらないこと"を先に覚える。

### 絶対にやってはいけない（お客様に頼まれても断る）
| NG行為 | なぜダメか | 断り方 |
|---|---|---|
| **サクラ・自作自演の口コミ** | 発覚で全口コミ削除・停止 | 「それはお店の資産を壊すので、うちは一切やりません」 |
| **「口コミくれたら割引」（対価）** | ガイドライン違反 | 「特典と口コミは切り離します。それが安全です」 |
| **満足客だけに口コミ依頼**（レビューゲーティング） | 2026年ルールで停止対象 | 「全員に均等にお願いする形にします」 |
| **店名にキーワード詰め込み**（例：「大宮ネイル｜安い｜人気」） | プロフィール停止の直接原因 | 「正式名称のままにします。停止リスクなので」 |

**これらを断ることが、逆に信頼になる。** 「ちゃんと分かってる業者だ」と思われる。

### 薬機法・広告規制（業種で効く）
- **美容・エステ・整骨・歯科**は「治る」「痩せる」「シミが消える」など**効能の言い切りがNG**。
- 我々のツールは**投稿文を機械で自動チェックして危ない表現を止める**（＝素人でも事故らない仕組みがある）。
- だから「文章はAIが作るけど、**人が薬機法を確認してから出す**」＝これが安全＆売りになる。

### 成果保証しない（これも守り）
- 「順位◯位にします」「問い合わせ◯件増やします」は**言った瞬間アウト**（景表法リスク＋そもそも守れない）。
- 代わりに保証するのは**「作業をちゃんとやること」**（投稿◯本・返信・レポート）。ここは100%守る。

## 4. お客様の課題にどう訴求するか（刺さる営業の言葉）

### 訴求の黄金パターン：「もったいない」を見せる
お客様は「順位を上げたい」ではなく「**もっとお客に来てほしい**」と思っている。だから：

> 「今、"大宮 ネイル"で検索すると、御社も一覧に出ています。でも**写真が1枚で、口コミの返信もない**状態だと、隣の店と並んだとき選ばれにくいんです。せっかく見つけてもらえてるのに、**もったいない**ですよね。そこを整えるのが私たちの仕事です。」

→ **提案書ジェネレーターの「現状診断」がこの"もったいない"を自動で可視化**してくれる。これが最強の営業ツール。

### 課題別・刺さる一言集
| お客様の悩み | 刺さる訴求 |
|---|---|
| 「忙しくて投稿する時間がない」 | 「**投稿は丸ごとお任せください。**企画から掲載まで、承認だけで済みます」 |
| 「口コミに何て返せばいいか分からない」 | 「**返信文はこちらで作ります。**低評価にも角が立たない返し方で」 |
| 「前の業者に騙された/効果なかった」 | 「**順位は保証しません。**代わりに、やった作業を毎月数字でお見せします」 |
| 「Googleとか分からない」 | 「**設定も運用も全部こちらで。**最初の30分だけ、方針を一緒に決めさせてください」 |

### 信頼を勝ち取る「正直トーク」
できることだけでなく**「できないこと」を先に言う**と、逆に信頼される：
> 「写真の撮影はやりません。順位保証もしません。サクラも絶対やりません。**その代わり、御社の資産を壊さず、比較で選ばれる状態を正直に育てます。**」

## 5. 受注したら、実際どう回すの？（オペレーションの全体像）

素人が一番不安な「受注後」。流れはたった5ステップ：

```
① 提案（マップのリンク→提案書ジェネレーターで自動作成）
   ↓
② 契約（竹プラン中心。最低1年・2年で初期無料）
   ↓
③ オンボーディング面談（30〜60分・ここだけ人がやる）
   ・GBP権限をもらう
   ・投稿テーマ・口コミの方針を一緒に決める（＝運用の"憲法"）
   ・「★4-5の返信は自動でOK？」など自動化の同意をとる
   ↓
④ 日々の運用（ほぼAI＋承認だけ）
   ・投稿・口コミ返信はAIが下書き→人が承認→掲載
   ・危ないもの（低評価・薬機法アウト）だけ人に上がってくる
   ↓
⑤ 月次レポート（自動生成→送付）
   ・「今月これだけやりました」＋数字を見せる＝解約されない
```

### なぜ素人でも回せるのか
- **判断は全部ルール化済み**（`meo-escalation-rules.md`）。★4〜5でNGワード無しの口コミは自動返信、★1〜3や「やけど」「返金」等の危険ワードだけ人が対応。
- だから毎日大量の承認をする必要がなく、**本当に人が見るべき数件だけ**でいい。

## 8. 素人が最初に暗記する「3つの禁句」と「3つの決め台詞」

**禁句（言ったら事故る）**
1. 「順位を◯位に上げます」
2. 「口コミ増やすので割引つけましょう」
3. 「店名にキーワード入れましょう」

**決め台詞（これで売れる）**
1. 「比較されたとき"選ばれる受け皿"をつくります」
2. 「投稿も返信も丸ごとお任せ、あなたは承認だけ」
3. 「順位は保証しませんが、やった作業は毎月数字でお見せします」

この6つを覚えれば、明日から話せる。
