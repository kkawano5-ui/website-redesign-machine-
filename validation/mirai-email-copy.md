# みらい編集｜コールドメール文面集（値段は伏せる・相談ファースト）

> 目的：中小企業オーナーに「売り込み」ではなく「ちょっと相談してみるか」と思わせ、**無料診断/15分相談**へ進めてもらう。
> 原則：**①値段は本文に書かない（伏せる）②1メール＝1メッセージ＝1CTA ③短く（スマホで5秒で読める）④業種・地域を差し込んで“あなた宛”にする ⑤誇張しない**。
> A/Bで「いろんな数字」を出すため、**件名・本文・CTAをそれぞれ複数用意**。最初は件名のA/Bから回すのが鉄則（最も差が出る）。

---

## 0. 差し込み変数（送信ツール側でマージ）

| 変数 | 例 | メモ |
|---|---|---|
| `{{会社名}}` | 〇〇工務店 | 必須。無いと一斉送信感が出る |
| `{{業種}}` | 飲食店 / 美容室 / クリニック / 工務店 / 整体院 | 件名・冒頭で使う |
| `{{地域}}` | 福岡市 / 〇〇エリア | MEO訴求に効く |
| `{{担当者}}` | 河野様（無ければ「ご担当者様」） | 取れたら開封・返信が上がる |
| `{{差出人}}` | 河野 邦彦 | 署名 |

> ⚠️ 値段（月額・単価）は**1文字も入れない**。聞かれたら「お会いした時 or 診断後に、御社に必要な分だけ」で返す。

## 1. 件名（A/B/C/D｜まずここを回す）

> コツ：数字・地域・業種で“自分ごと化”。煽り(!や【限定】)・記号過多は迷惑メール判定とスルーの元。**14〜26文字目安**、スマホで切れない長さ。

- **A（コスト軸）**：`{{地域}}の{{業種}}さんへ｜固定費、見直せる余地あるかも`
- **B（集客軸）**：`Googleマップで「{{地域}} {{業種}}」、上位に出ていますか？`
- **C（ワンストップ/相談軸）**：`{{会社名}}様｜集客・採用・コスト、まとめて1回相談しませんか`
- **D（軽い・好奇心軸）**：`検針票1枚だけ、見せていただけませんか`

> D は「電気代診断」の入口に直結。返信率は高いが対象を選ぶ。A/Bを主軸、C/Dを当て馬で。

## 3. フォローアップ（返信が無い人へ・自動化）

> コールドは1通で決まらない。**2〜3通目で返信の半分以上が来る**のが普通。同一スレッドに返信する形（Re:）で。各回の間隔は3〜4営業日。

### フォロー1（3〜4日後）
```
件名：Re:（1通目と同じ）

{{会社名}} {{担当者}}

先日の件、そっと上にあげさせてください。
「今は不要」でも全く問題ありません。その場合は無視していただければ、
こちらからの連絡はこれで止めます（しつこくしません）。

もし少しでも気になる点が1つでもあれば、その1点だけでもお答えします。
{{差出人}}
```

### フォロー2（さらに4〜5日後・最後）
```
件名：Re:（同上）

{{会社名}} {{担当者}}

最後のご連絡です。今のタイミングでなくても、
「集客・採用・コストで困ったら、まとめて聞ける会社がある」とだけ
頭の片隅に置いていただければ十分です。

必要になったタイミングで、この返信か 070-8311-7053 にご連絡ください。
ありがとうございました。{{差出人}}
```

## 5. 送信前チェック（毎回）

- [ ] 値段（月額・単価・円）が本文に**入っていない**
- [ ] `{{会社名}}`〜`{{業種}}`〜`{{地域}}`が正しく差し込まれている（マージ崩れ＝即ゴミ箱）
- [ ] CTAは1つだけ（「返信して」or「マップ見て」など）
- [ ] リンクは1本、UTM付き（`utm_content`を文面ごとに変える＝どの文面が効いたか分かる）
- [ ] 署名に**会社名・氏名・連絡先・配信停止の方法**がある（特定電子メール法／後述）
- [ ] 件名に【】！￥などの記号・煽りが無い
- [ ] スマホ表示で1スクロールに収まる

## 6. 法令メモ（特定電子メール法）※必読・送信前に

- 営業メールには **①送信者の氏名/名称 ②配信停止の方法** の表示が必須 → 上記署名で対応済み。
- 原則オプトインだが、**「会社のサイト等で公表しているメールアドレス（事業用）」宛で、かつ受信拒否の表示が無い**場合は送付できる例外がある。→ **企業の公開問い合わせ/代表アドレス宛に限定**し、個人アドレスや拒否表明先には送らない。
- 「配信停止」と返信が来たら**即リスト除外**（再送しない）。除外管理は trで列を持つ（`mirai-leads-tracker.csv` の `status=optout`）。
- 不安な点は最終的に専門家確認を推奨（ここはマーケ設計であり法的助言ではない）。
