## 1. プロジェクト要約（AI実装者向け）
- ゴール: 地方中小企業向けに、信頼感・清潔感・現代感のあるコーポレートサイトを制作する。
- 優先順位: ①信頼性 ②分かりやすさ ③問い合わせ導線 ④表示速度。
- 禁止事項: 既存サイト文言のコピペ、誇大表現、著作権リスク画像の使用。

## 2. 会社・市場理解
### 2-1. 会社概要
- 山梨県北杜市を中心に戸建てリフォーム・外壁塗装・水回り改修を行う地域密着企業。
- 創業28年、累計施工実績3,200件。

### 2-2. 既存サイトの課題
- スマホ表示で文字が小さく、問い合わせボタンが見つけづらい。
- 施工事例の更新が止まっており、信頼性が伝わりにくい。

### 2-3. ターゲット顧客
- 北杜市・韮崎市・甲斐市在住の40〜70代戸建てオーナー
- 古くなった水回りや外壁を安心して相談したい層

## 3. 企画方針
### 3-1. サイトコンセプト
- 地元で長く選ばれてきた安心感を、実績と担当者の顔が見える情報設計で伝える。

### 3-2. ファーストビュー訴求案
- 目的: 訪問後10秒以内に「誰向け / 何が強みか / どう行動すれば良いか」を理解できる状態を作る。
- 必須要素:
  - キャッチコピー（地域性 + 提供価値 + 実績）
  - サブコピー（顧客課題と解決像）
  - ヒーロー画像（施工現場/スタッフ/完成写真）
  - 信頼補強（対応エリア、施工件数、創業年、資格）
  - 主CTA（無料相談/見積依頼）
- 訴求テキスト案:
  - 北杜市の住まいを、28年の経験で長持ちリフォーム
  - 現地調査・見積無料。しつこい営業なしで安心相談

### 3-3. CTA案
- 無料見積を依頼する
- 施工事例をもっと見る
- 電話で相談する（平日9:00-18:00）

## 5. デザイン/UX要件
### 5-1. デザイントーン
- 白とネイビー基調で清潔感・信頼感を演出
- 実写写真中心で誠実さを伝える
- 余白: セクション上下は72〜96px、本文行間は1.8前後で可読性を担保。
- タイポグラフィ: 見出しは700、本文は16〜18px。
- 色設計: ベース70% / サブ20% / アクセント10%。CTAカラーは固定。
- UI方針: 控えめな角丸（6〜10px）、アニメーションは必要最小限。

### 5-2. 表現上の注意
- 絶対に直る、100%満足など断定表現
- 他社を貶める比較表現

## 6. 実装仕様（Claude / Cursor / Codex へそのまま渡す）
### 6-1. 技術スタック
- Astro + Tailwind CSS
- モバイルファースト（375px基準）
- semantic HTML + aria属性
- Lighthouse目標: Performance/Accessibility/Best Practices/SEO 各90+

### 6-2. 実装スコープ
- 主要ページ: Top / サービス / 事例 / 会社案内 / お問い合わせ
- 全ページに主CTA + 補助CTA
- 再利用コンポーネント: Hero / Section / CTA / Footer / Header
- 差し替え可能領域（文言・画像・連絡先）をコメントで明示

### 6-3. コンテンツ作成ルール
- ダミーテキストは業種に沿った自然な日本語で生成
- 断定的No.1表現など根拠不明の優良誤認表現は避ける
- 画像は著作権フリー素材またはプレースホルダーを使用

### 6-4. 追加実装指示
- 各サービス紹介に標準工期と費用目安レンジを入れる
- 施工事例はビフォー/アフター比較をカードUIで表示する
