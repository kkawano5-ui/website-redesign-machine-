# 法人AI研修 提案資料（PPT・2バージョン）

営業マニュアル（Notion 00〜11）の思想をベースに、対象別に **中小企業版／大企業版の2パターン** で作成する提案資料（PowerPoint）です。

## ファイル
- `ai-training-proposal-sme.pptx` … **中小企業版**（全24スライド・16:9）★現行
- `ai-training-proposal-enterprise.pptx` … 大企業版（予定。浸透ロードマップ＋「ツール導入でなく社員教育が定着のカギ」を主軸）
- `deck_common.py` … 共通デザイン部品（THEME定数・ヘルパー）。両版が共有
- `build_sme.py` … 中小企業版ビルダー（`python3 build_sme.py` で再生成）
- `storyline.md` … ストーリーライン解説・設計方針・助成金の準拠表現・使い方

## 方針（2バージョン）
- **中小企業版**：具体・中小寄り。「取り残されたくない」「言い訳外し」を入口に、中小の助成率（75%）・実質負担（約98,675円）・取引先の先行（外圧）を前面に。
- **大企業版（予定）**：全社浸透ロードマップ（フェーズ別の成功パターン）を中核に、「ツールの導入ではなく“社員教育”こそが定着のカギ」というメッセージを明確化。

## 使い方
- `.pptx` を **PowerPoint で開く**（フォント＝游ゴシック / Yu Gothic）。
- 表紙の宛名「◯◯◯◯株式会社 御中」「ご提案日」は商談ごとに差し替え。
- 資料は“見せてヒアリングを引き出す”ための素材です。質問・記入はスライドに入れず、口頭で行う構成です。

## 再生成
```
pip install python-pptx
python3 build_sme.py    # → ai-training-proposal-sme.pptx
```
色・フォント・余白は `deck_common.py` の THEME 定数で一括変更できます（メイリオ/BIZ UDP等への差し替えも1〜2行）。

詳細は `storyline.md` を参照。
