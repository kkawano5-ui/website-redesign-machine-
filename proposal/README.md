# 法人AI研修 提案資料（改訂版）

営業マニュアル（Notion 00〜11）の思想に沿って、提案のストーリーラインを再設計し、
BtoB向けにシンプル＆洗練したデザインで刷新した提案デッキです。

## ファイル
- `index.html` … 提案デッキ本体（自己完結のHTML。全21枚＋付録3枚）
- `storyline.md` … ストーリーライン解説（背骨・各スライドの意図・マニュアル根拠・使い方）
- `vercel.json` … 静的公開用設定

## 使い方
- ブラウザで `index.html` を開く。
- `← →`（またはクリック）でスライド送り。`P` キーで印刷／PDF書き出し（1スライド＝1ページ・16:9）。
- URL末尾に `#3` のようにページ番号を付けるとそのスライドへ直接移動。
- 表紙の宛名「◯◯◯◯株式会社 御中」と日付は、商談ごとに `index.html` の該当箇所を差し替える。

## 設計の要点
- ストーリー：取り残されたくない → 課題の再定義 → 提供価値と証拠（“誰も使わない”を潰す）→ 中身と進め方 → 今やる理由と費用 → 背中押し・クロージング。
- 会社概要・監修者・実績は冒頭から中盤の「信頼の裏づけ」位置へ移送（独演を回避）。
- 価格は必ず「300,300円 → 助成金で実質約98,675円」の実質負担で提示（値引き訴求はしない）。
- 数値・事例はすべて提供資料の実データ。社外秘（KPI・IS/FS運営・架電台本）は非掲載。

詳細は `storyline.md` を参照。
