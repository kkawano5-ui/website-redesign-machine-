# 法人AI研修 提案資料（PPT版）

営業マニュアル（Notion 00〜11）の思想をベースに提案ストーリーラインを再設計し、
BtoB向けにシンプル＆洗練したデザインで作成した提案資料（PowerPoint）です。

## ファイル
- `ai-training-proposal.pptx` … 提案資料本体（全23スライド・16:9）
- `storyline.md` … ストーリーライン解説（構成・意図・デザイン方針・助成金の準拠表現・使い方）
- `build_pptx.py` … 上記pptxの生成スクリプト（python-pptx）。テーマ定数を変えて再生成・編集可能

## 使い方
- `ai-training-proposal.pptx` を **PowerPoint で開く**（フォント＝游ゴシック / Yu Gothic）。
- 表紙の宛名「◯◯◯◯株式会社 御中」と「ご提案日」は、商談ごとに差し替えてください。
- 資料は“見せてヒアリングを引き出す”ための素材です。**質問・記入はスライドに入れず、口頭で**行う前提の構成にしています。

## 再生成（任意）
```
pip install python-pptx
python3 build_pptx.py   # 同フォルダに ai-training-proposal.pptx を再出力
```
色・フォント・余白は `build_pptx.py` 冒頭の THEME 定数で一括変更できます。

## 設計の要点
- 構成：会社紹介・実績で信頼を先に → 現状/課題（規模問わず）→ 提供価値と証拠 → 中身・進め方・費用 → クロージング。
- 実績は「研修・セミナー導入200社超」ほかを前面に。事例は“規模・業種を問わず成果”の証拠として提示。
- 価格は「助成金活用時の自己負担イメージ（目安）」として、断定・保証を避けた準拠表現で提示。
- 社外秘（KPI・IS/FS運営・架電台本）は非掲載。

詳細は `storyline.md` を参照。
