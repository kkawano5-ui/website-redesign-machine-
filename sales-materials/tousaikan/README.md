# 陶彩館 営業資料（AI研修 提案デモ）

陶彩館（オーヤシマ株式会社／和歌山・泉佐野のリフォーム専門店）向けに、
**「AI研修を提案・実演するためのデモ資料」** です。
AIに会社の公開情報を学習させて自動生成した営業資料で、
「貴社の営業担当もAIでこのレベルの資料を作れるようになります」を示すのが目的です。

## 成果物

| ファイル | 内容 |
|---|---|
| `陶彩館_営業資料_AIデモ.pptx` | **本番** PowerPoint（全15ページ・16:9ワイド） |
| `build_pptx.py` | 上記PPTXの生成スクリプト（再生成・改変用） |
| `sales-deck.html` | 同内容のHTML版（ブラウザ閲覧・印刷用の予備） |

## 構成（全15ページ）

1. 表紙
2. 私たちの想い（経営理念）
3. 会社概要
4. 選ばれる3つの理由
5〜7. 理由①実績／②大型ショールーム／③自社施工・ワンストップ
8. 取扱い商材ラインナップ
9. 取扱メーカー（タカラ／TOTO／Panasonic）
10. ご相談から完成までの流れ
11. 保証・アフターサービス
12. 施工事例
13. 店舗・対応エリア
14. お問い合わせ（CTA）
15. **種明かし** ―「この資料、AIが御社サイトを学習して作りました」（研修提案の決め）

## 再生成・編集

```bash
pip install python-pptx
python3 build_pptx.py   # 陶彩館_営業資料_AIデモ.pptx を出力
```

PowerPoint で直接編集も可能。文言・数値・配色は `build_pptx.py` 上部のパレットと
各スライド定義から一括変更できます。日本語フォントは Meiryo を指定しています。

## 配布前に差し替えたい項目（公開情報ベースのサンプルのため）

- [ ] 電話番号・Web予約URL（14ページの「○○」部分）
- [ ] 会社概要の数値（資本金・従業員数・正確な施工件数・年商など）
- [ ] 施工事例の実写真・Before/After（`tousaikan.com/case` の許諾済み実例）
- [ ] お客様の声（11ページ：許諾を得た実例へ）
- [ ] 「和歌山県売上No.1」の出典・対象年

## 研修での使い方（AI活用のヒント）
