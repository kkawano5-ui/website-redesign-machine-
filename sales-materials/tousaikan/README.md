# 陶彩館 営業提案資料（AI研修サンプル）

和歌山・泉佐野のリフォーム専門店「陶彩館」（オーヤシマ株式会社）の
**営業担当者がそのまま使える提案資料**です。AI研修の成果物サンプルとして、
公開情報をもとに会社の強み・商材・特徴を整理し、14ページのスライドにまとめています。

## ファイル

| ファイル | 内容 |
|---|---|
| `sales-deck.html` | 営業提案スライド（全14ページ・印刷でPDF化可） |

## 使い方

1. `sales-deck.html` をブラウザ（Chrome / Edge 推奨）で開く
2. **印刷 → 送信先「PDFに保存」→ 用紙「A4・横向き」** で配布用PDFが作れます
3. お客様への提案、ショールーム来店時の説明資料として活用してください

## 構成（全14ページ）

1. 表紙
2. 私たちの想い（経営理念）
3. 会社概要
4. 選ばれる3つの理由（概要）
5. 理由① 確かな実績（50年・55,000件超・県内No.1）
6. 理由② 体感できる大型ショールーム（約800坪）
7. 理由③ 自社施工・ワンストップ体制
8. 取扱い商材ラインナップ
9. 取扱メーカー（タカラ／TOTO／Panasonic）
10. ご相談から完成までの流れ（6ステップ）
11. 保証・アフターサービス
12. 施工事例
13. 店舗・対応エリア
14. お問い合わせ（CTA）

## 営業担当者へ：配布前にここを差し替え

この資料は**公開情報ベースのサンプル**です。提案で使う前に以下を最新の正確な情報へ更新してください。

- [ ] 電話番号・Web予約URL（14ページの「○○」部分）
- [ ] 会社概要の数値（資本金・従業員数・正確な施工件数・年商など社内の最新値）
- [ ] 取扱メーカー・加盟ショップの最新状況
- [ ] 施工事例の写真とBefore/After（`tousaikan.com/case` の実例・お客様許諾済みのもの）
- [ ] お客様の声（11ページ：許諾を得た実例へ）

## AI活用のヒント（研修向け）

このようにAIへ依頼すると営業資料づくりを効率化できます。

- 「お客様の家が**築30年・4LDK**の場合に響く事例3つを、この資料から選んで紹介トークを作って」
- 「この資料の内容を、**チラシ1枚（A4表裏）**用に要約して」
- 「**60代のご夫婦**向けに、バリアフリーと断熱を中心にした提案ページを追加して」
- 「メーカー比較（タカラ／TOTO／Panasonic）の**説明トークスクリプト**を作って」
