# 店舗写真の置き場所

各店舗の画像を `public/shops/{slug}/` に配置します。未配置でも上質なプレースホルダー枠が表示されます。

## 使用ファイル名（shops.ts の参照に合わせる）
- `hero.jpg` … ヒーローのメイン写真
- `ogp.jpg` … SNSシェア用（OGP）
- `menu-1.jpg` 〜 `menu-5.jpg` … メニュー写真
- `gallery-1.jpg` 〜 … ギャラリー写真

## 推奨
- 横長は 1600×1067 以上、正方は 1200×1200 以上
- 実店舗写真がない間は配置不要（プレースホルダーが出ます）
